# Languages ≠ Flags

## Duolingo redesign | Personal project, 2020

## The quick version

Languages don't map to flags. I redesigned Duolingo's course selection to fix a system that excludes, confuses, and can't follow its own logic.

## Before I was a designer, I was an international studies student and a longtime Duolingo user

I noticed something that bothered me: Duolingo represents language courses with flags. But which flag represents Spanish — Spain? Mexico? The 19 other countries? Which flag gets Portuguese — Brazil or Portugal? The current system can't follow logic because languages and flags don't map cleanly. It excludes, confuses, and the CEO has acknowledged it's flawed.

## Curiosity and a problem worth solving — that was enough to start

After researching the problem, illustrating flags for 20+ countries, and iterating through possible solutions, I landed on something simple: use ISO-639 language codes for course selection (standardized, uniform length, immediately implementable) and repurpose flags to highlight cultural diversity within course content — stories, roleplay scenarios, loading screens.

## One change removes the problem. The other turns flags into an asset.

Duolingo, if you're reading this — just do it. I'll wait.
